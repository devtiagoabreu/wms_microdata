Como Acessar na Rede

  1. Certifique-se que todos os computadores estão na mesma rede Wi-Fi
  2. No computador que vai acessar, abra o navegador (Chrome, Firefox, Edge)
  3. Digite uma das URLs acima na barra de endereços
  4. Pressione Enter para acessar o gerador de etiquetas
  5. Para celular, escaneie o QR Code ou digite a URL manualmente
